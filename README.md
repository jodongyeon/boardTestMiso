# boardTestMiso
게시판 formsubmit방식과 AJAX비동기방식으로 다른 방식의 두 가지 게시판이다.




웹 기능  
              1. 게시판 CRUD
              2. 파일 등록, 파일 다운로드
              3. 페이징
              4. 검색
              5. list 갯수 변경하는 selectbox             
              
#개발환경
    -Web
    * SERVER : apache-tomcat-8.5.34
    * DATABASE : MariaDB
    * IDE : Eclipse
    * Language & Framework : Java, Springframework, AJAX, jquery
